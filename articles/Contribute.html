Contribute

How to contribute to DeMON wiki?

1. Join DeMON+BioFINDER Github organization

2. Get access to DeMONLabWiki repo

3. Create a new branch

4. Edit contents

5. Submit Pull Request (PR)

6. Review & merge your PR

7. Update your own branch
